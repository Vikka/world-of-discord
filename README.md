# world-of-discord
World of Discord is a MMO RPG you can play within Discord.

# requiered rights 
To create a link that allow any server owner to add the bot to a server, you need to check thoses rights:
- [x] Manage Channels,
- [x] View Channels,
- [x] Send Messages,
- [x] Manage Messages,
- [x] Read Message History
- [x] Add Reactions
